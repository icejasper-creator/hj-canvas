操作指南 How to Use
5个工具 · 选工具 → 在画布操作
🖱
选择 Select
点任何东西（墙/区域/设备/门窗/连线）→ 选中 → 拖动移动 · ✕删除 · 空白拖动=平移画面。万能编辑工具。
🔄
旋转图纸 Rotate Plan
点一下把整张图旋转90°（不是CCTV角度）。导入的PDF方向歪了可以转正。
⬜
区域 Zone
拖拉画一个矩形框标记空间 → 自己取名（如客厅、厨房、停车场）。点框内部选中可移动/改大小/删除。自动显示尺寸和面积。
〰
智能线 Smart Wire
点设备1 → 点设备2，系统自动配线：Analog→RG59、IPC→CAT6、传感器→485、锁→电源。超100米自动改光纤Fiber。
📄
导入图纸 Import PDF
右侧 📄 → 选客户PDF/图片当背景 → 🔒锁定防误移 → 校准比例 → 在上面放设备画线（主要流程）
✏️
画墙 Draw Wall
没有PDF时手画墙（自动45°吸附）· 点已有墙→红色→再点删除 · 墙上显示米数
🚪
门禁/门磁/窗磁（用 📍 设备放）
门窗安防不再是独立工具。用 📍 设备 → 选 门禁/电锁/门窗磁/Exit按钮/Autogate自动门 → 放在 PDF 的门窗位置上即可。
📍
设备 Device
先在底部选设备类型 → 点画布放置 → 手指不放可直接拖到准确位置
〰
连线 Wire
点第1个设备 → 点第2个设备 → 自动识别线型并连接 · 点已有连线→删除 · 连完自动回画墙
📌 选中设备后（操作环）
✕删除设备（含连线）
⟳旋转视野（仅摄像头）· 拖动调方向
✎编辑（名称 / 安装方式 / 高度 / 供电）
↔直接拖设备本体 = 移动位置
🖐️ 通用手势
👆👆双击空白 = 缩放适应全图
🤏两指捏合 = 缩放 + 平移
↩右上角按钮 = 撤销 / 适应 / 清空
